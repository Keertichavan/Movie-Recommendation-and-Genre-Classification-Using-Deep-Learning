# Movie Recommendation and Genre Classification Using Deep Learning

## Project Overview

This project implements a Movie Recommendation and Genre Classification System using Deep Learning techniques. The system analyzes movie overview text, classifies movie genres, and recommends similar movies based on textual content.

## Features

* Data Preprocessing
* Exploratory Data Analysis (EDA)
* Genre Classification
* Movie Recommendation System
* Deep Learning Model Comparison

## Deep Learning Models Used

* Artificial Neural Network (ANN)
* Recurrent Neural Network (RNN)
* Long Short-Term Memory (LSTM)
* Gated Recurrent Unit (GRU)

## Technologies Used

* Python
* TensorFlow
* Keras
* Pandas
* NumPy
* Scikit-Learn
* Matplotlib
* Seaborn

## Dataset

The project uses:

* movies_genres.csv
* movies_overview.csv

## Workflow

1. Load Dataset
2. Data Cleaning
3. Exploratory Data Analysis
4. Text Preprocessing
5. Tokenization and Padding
6. Label Encoding
7. Train ANN, RNN, LSTM, and GRU Models
8. Compare Model Performance
9. Generate Movie Recommendations

## Results

The project compares the performance of multiple deep learning models and identifies the best-performing model for genre classification.
